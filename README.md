# HR Analytics Dashboard
## Project Overview
This project is an HR Analytics Dashboard created in Microsoft Power BI.  
The dashboard helps analyze employee data, attrition trends, workforce distribution, job satisfaction, and salary insights.
## Dashboard Features
- Total Employees
- Total Attrition
- Attrition Rate
- Active Employees
- Average Age
- Average Monthly Salary
- Employees by Age Group and Gender
- Attrition by Age
- Attrition by Department
- Attrition Rate by Job Role
- Job Satisfaction Rating
- Attrition by Education Field
## Tools Used

- Data Cleaning
- Data Visualization

## Purpose of the Dashboard
The purpose of this dashboard is to help HR teams understand employee attrition, workforce patterns, job satisfaction, and department-wise employee trends. It supports better decision-making using clear and interactive visuals.
## Key Insights
- Tracks overall employee strength and attrition.
- Shows attrition trends by age, department, job role, and education field.
- Compares employees by gender and age group.
- Displays job satisfaction rating by job role.
- Helps identify areas where employee retention can be improved.
